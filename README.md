## 项目环境
项目运行在Windows10，conda管理的Python 3.7.6。

项目包括但不限于使用了以下package：
* jupyter 
* numpy 
* matplotlib 
* scikit-learn

  
如果要安装环境，请在激活的3.7.6 conda环境中运行
```
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
```
